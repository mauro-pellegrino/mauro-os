# content/qa — Questions & Answers (raw fuel)

This is where the interview material lives: **Juan's questions + Mauro's transcribed voice-note answers**, one file per topic.

## The workflow (from `skills/content/visual-docs/juan-tutorial.md`)
1. Juan writes good B2B / agency questions worth a video or article answer.
2. Mauro answers them in voice notes.
3. The voice notes get transcribed, cleaned (fix transcription artifacts, keep Mauro's voice), and saved here as one Q&A file.
4. That file becomes source material for scripts, X articles, short-form, and lead magnets.

## Rules
- One file per topic. Name it by topic, e.g. `virality-reach-qa.md`.
- Top of every file: date, source, and what it's for (see the existing file as the template).
- Keep Mauro's voice. Only fix transcription garble, never rewrite meaning or invent numbers.
- If a question wasn't answered in the recording, mark it `[Not covered in this recording. To answer.]` instead of filling it in.
- This is **source material**, not a finished asset. Full video transcripts still go in `research/transcripts/maurojpelle/`.
