When writing about Meta's Andromeda update, entity IDs, or creative diversity, the load-bearing argument is that creatives must be **visually** different, not just different scripts or hooks. A new hook on the same template (same background, same framing, same person, reworded text) gets the same entity ID, so Meta treats it as one ad and flattens the variation. Visual difference (different format, setting, person on camera, e.g. a podcast clip next to a street interview next to a founder talking to his phone) is what earns a distinct entity ID and lets Meta deliver each creative to a different type of buyer. This also reframes the offer: producing genuinely (visually) different creatives at volume is a production problem, which is exactly what Growthub solves.

**Why:** Mauro flagged on 2026-06-04 that the Andromeda article was missing this; without it, "make more variations" sounds like "reword the hook," which the algorithm sees through.

**How to apply:** Every Andromeda or creative-diversity piece for [[project_lorenzo_x_account]] should carry the visual-difference point explicitly.
